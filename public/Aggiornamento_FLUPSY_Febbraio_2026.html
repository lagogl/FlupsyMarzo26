  4. Le ceste visualizzate saranno filtrate automaticamente
📅 Toggle Data Attivazione / Ultima Operazione
NUOVO Pulsante di scambio nella colonna data

Nella tabella, accanto alla colonna "Ult.Op" c'è ora un pulsante blu ⇄ che permette di alternare tra:

  • Ult.Op = Data dell'ultima operazione registrata
  • Attiv. = Data di prima attivazione del ciclo
Come usarlo
  1. Trova la colonna "Ult.Op" nell'intestazione della tabella
  2. Clicca sul pulsante blu ⇄ accanto al titolo
  3. La colonna cambierà in "Attiv." mostrando le date di attivazione
  4. Clicca di nuovo per tornare a "Ult.Op"

Nota: L'ordinamento funziona in base alla data attualmente visualizzata.

📊 Esportazione Google Sheets
NUOVO Pulsante "Google Sheets" arancione

Accanto al pulsante verde "Esporta Excel" c'è ora un pulsante arancione "Google Sheets".

Vantaggi del formato Google Sheets:

  • Date in formato corretto (YYYY-MM-DD) riconosciute automaticamente
  • Numeri senza separatori delle migliaia (pronti per calcoli)
  • Include anche la Data Attivazione come colonna separata
  • Compatibile con importazione diretta in Google Sheets
Come importare in Google Sheets
  1. Clicca il pulsante arancione "Google Sheets"
  2. Scarica il file CSV generato
  3. Apri Google Sheets e vai su File > Importa
  4. Seleziona il file CSV scaricato
  5. I dati saranno pronti con formati corretti!
